# 电子行业 80 家样本研究地图

## 结论

当前仓库适合做「卖方预期核验与预期差发现」，不适合直接批量产出 80 篇单公司长报告。研究应按「专题比较 → 单公司深挖 → 财报后复盘」推进：先完成 CCL／AI PCB、存储价格与嵌入式存储、先进封装三个专题，再从每个专题选 2～3 家进入单公司深研。

这份地图只规定研究边界、证据标准和排期，不重复已有的财务、估值与市场行为明细。

## 数据边界与证据标准

| 层级 | 当前来源 | 可支持的判断 | 不能替代的判断 |
| --- | --- | --- | --- |
| A 级 | 公司公告、交易所披露 | 已披露的收入、利润、现金流、资产负债与正式指引 | 尚未披露季度的经营结果 |
| B 级 | `daily_basic`、`normalized_fundamentals`、`margin_detail` 与同花顺数据 | 估值历史分位、Q1 财务质量、热度、涨停、资金流与融资变化 | 客户订单、产能利用率、产品收入占比与价格传导 |
| C 级 | [原始研报](../archive/original-analytic-report.md)、同花顺概念快照 | 待验证的 26Q2 预期与市场叙事 | 已确认的公司经营事实 |

- 数据截止口径以 [增强初筛报告](enhanced-screening-report.md) 首页为准：财务为 2026-03-31，估值与融资数据使用各自的最近可用交易日。
- 当前尚未有 26Q2 正式财务披露。因此专题报告只能判断「Q1 是否支持原预测、Q2 需要什么条件才能兑现」，不能把 26Q2 预测写成已实现结果。
- `hsgt_top10_days` 只记录进入沪深港通每日十大成交的次数；它不代表净买入、持仓变化或投资者身份。市场行为只能用于确定核验优先级。

## 研究流程

```text
原始预期 → 初筛与市场行为 → 专题横向比较 → 单公司深研 → 财报实际与原预测复盘
```

每个专题先输出一张可比较的公司矩阵，再选 2～3 家写单公司深研。不得因 P0／P1、热榜天数或研究价值分单独决定研究顺序。

### 单公司进入深研的三项门槛

| 门槛 | 要回答的问题 | 通过标准 |
| --- | --- | --- |
| 业绩影响 | 主题是否能影响公司收入、毛利或净利润？ | 能明确指出产品、客户、价格、产能或成本路径 |
| 预期分歧 | 市场定价与可验证经营变量是否存在缺口？ | 原预测、Q1 结果、估值和市场行为不能相互解释 |
| 可验证性 | 未来 3～6 个月能否找到反证数据？ | 已列出公告、价格、订单、库存或产能等观察项 |

## 第一轮专题排期

| 顺序 | 专题 | 重点比较 | 建议先做的公司深研 | 当前专题底稿 |
| --- | --- | --- | --- | --- |
| 1 | CCL／AI PCB | 电子布与 CCL 涨价能否传导至 PCB 毛利；AI 产品与传统业务的结构差异 | 生益科技、沪电股份、广合科技 | [专题底稿](topic-ccl-ai-pcb.md) |
| 2 | 存储价格与嵌入式存储 | 直接存储价格、库存成本与合封存储业务分别如何影响利润 | 兆易创新、北京君正、东芯股份 | [专题底稿](topic-storage-price.md) |
| 3 | 先进封装 | 稼动率、先进封装收入占比、客户导入和产能利用率是否支撑预测 | 长电科技、通富微电、甬矽电子 | [专题底稿](topic-advanced-packaging.md) |

北方华创列为下一阶段的设备专题候选。设备收入确认、在手订单与晶圆厂资本开支的验证逻辑不同于算力芯片和封测，不应为了覆盖面把它们合并到上述三个专题。

## 单公司报告的固定交付

沿用 [公司研究待办清单](company-analysis-todo.md) 的九节模板，并在「一页结论」固定增加以下内容：

1. 原报告主题对收入、毛利和增量利润的真实相关度：高／中／低，及其证据来源。
2. 26Q2 预测的收入桥、毛利率桥和净利润桥；无法拆分时必须写明缺失字段。
3. 当前市值在 30 倍、50 倍和行业合理区间下隐含的利润要求，并与原预测比较。
4. 一个最早可观察的高频指标，以及产业、公司、财务、估值和交易结构五类反证条件。
5. 资料分级：公告与交易所披露为 A 级，持久化市场数据为 B 级，研报和概念标签为 C 级。

## 复盘机制

26Q2 披露完成后，每家进入深研的公司必须补一张「原预测—实际—差异归因」表：

| 项目 | 原预测 | 实际披露 | 差异 | 归因 | 是否改变后续判断 |
| --- | --- | --- | --- | --- | --- |
| 收入 |  |  |  | 出货量／ASP／产品结构／确认节奏 |  |
| 毛利率 |  |  |  | 涨价／成本／折旧／产品结构 |  |
| 归母净利润 |  |  |  | 费用／汇兑／投资收益／税率 |  |

这套复盘表是将研究从「叙事整理」变为可累积判断的最低要求。
