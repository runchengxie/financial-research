## 使用方式

- `[ ]` 表示尚未完成；完成研究后改为 `[x]`，并在末尾附上产出文件路径和分析日期。
- [初筛清单](company-analysis-initial-screen.md) 已记录 80 家的基础数据核对；该文件的 `[x]` 不代表本清单的完整研究任务完成。
- `P0` 为报告正文中被明确称为「表现优异」的公司，建议先由研究负责人梳理；`P1` 为表格中其余覆盖公司。优先级是对原文措辞的整理，不构成投资建议。
- 每份公司研究至少应覆盖：业务与产品、最近财报对原预测的验证、需求/价格/产能驱动、竞争位置、主要风险、待验证假设及来源链接。原表的 26Q2 数字为当时预测，不能当作最新事实直接引用。

## 建议的单公司交付模板

```md
# {公司名}（{证券代码}）深度研究

## 0. 一页结论
- 当前结论：看多 / 中性 / 看空 / 尚无结论（附置信度：高 / 中 / 低）
- 核心分歧：市场认为 X，但我认为 Y，因为 Z
- 未来 3-6 个月关键催化：
- 最大反证条件：出现什么情况说明逻辑错了
- 估值是否已计入增长：是 / 否 / 无法判断
- 市场共识：{一句话总结市场主流观点}
- 预期差来源：{你和市场判断不同的地方}

## 1. 业务拆解
- 主营业务与收入结构
- 毛利结构
- 主要客户与供应商
- 产品所在产业链位置
- 与原报告主题的真实相关度

## 2. 原报告预测验证
- 原报告预测内容
- 最新财报是否支持
- Q1 到 Q2 连续性判断
- 数据冲突点

## 3. 行业与周期位置
- 行业供需
- ASP / 成本 / 库存
- 产能与稼动率
- 竞争格局
- 上下游议价能力

## 4. 财务桥
- 收入桥：出货量 × ASP × 产品结构
- 毛利率桥：涨价 - 成本上涨 - 折旧 - 产品 mix
- 净利润桥：毛利率 + 费用率 + 一次性因素
- 现金流与营运资本：经营 CF / 净利润，应收/存货趋势
- 一次性因素剔除：汇兑 / 补贴 / 投资收益 / 税务转回

## 5. 估值
- 历史估值分位（参考 enhanced-screening-report.md）
- 可比公司估值
- 隐含利润 / 增长反推
- 乐观 / 中性 / 悲观情景

## 6. 市场行为
- 股价涨幅与成交活跃度
- 沪深港通 Top10 交易活跃度 / 融资 / 资金流方向
- 拥挤度评分
- 背离信号（参考 company-hotspot-analysis.md §4）

## 7. 反证与风险
- 产业反证
- 公司反证
- 财务反证
- 估值反证
- 交易结构反证

## 8. 资料来源
- 公司公告（A 级）
- 交易所披露（A 级）
- 行业数据（标注具体来源与可信度）
- 第三方数据库（B 级，标注接口）
- 研报 / 纪要（标注出处与可信度）
```
