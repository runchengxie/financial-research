## 读法

- PE/PB 分位基于最近 3 年历史，`>p90` 表示当前估值处于近 3 年最高的 10% 区间。
- 现金流质量 = 经营活动现金流净额 / 归母净利润。Q1 为单季，FY 为全年。比率持续 < 0.5 需警惕利润质量。
- 融资拥挤度 = 融资余额 / 流通市值。> 3% 为较高杠杆，> 5% 为拥挤。
- 背离标签表示市场行为存在矛盾（如热度高但资金流出），共识标签表示多维度信号一致。
- 沪深港通 Top10 天数仅表示进入每日十大成交的次数，不表示净买入、持仓变化或投资者身份。

### PE/PB 背离分析

PE 与 PB 分位不一致时，估值信号的含义不同：

- **多重扩张**（PE 和 PB 均 > p90）：市场定价繁荣，估值容错空间极小。
- **利润压缩型**（PE > p90 但 PB 正常）：高 PE 来自利润处于周期底部，利润恢复后 PE 自然回落，可能是低估信号。
- **资产重估型**（PB > p90 但 PE 正常）：资产已被重估但利润尚未跟上，需验证利润能否兑现。
